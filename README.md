# README - ReformaT Venezuela 🏠

> **La página web de remodelaciones más premium de Venezuela**

## 🚀 Tech Stack

- **Framework:** Next.js 14+ (App Router) + TypeScript
- **Estilos:** Tailwind CSS con dark mode premium
- **Animaciones:** Framer Motion (scrollytelling, fades, parallax)
- **3D Interactivo:** React Spline (modelos 3D inmersivos)
- **Iconos:** Lucide React

## 📁 Estructura

```
src/
├── app/
│   ├── layout.tsx         # Root layout con metadatos SEO
│   ├── page.tsx           # Homepage heroica
│   ├── contacto/page.tsx  # Formulario + info
│   ├── galeria/page.tsx   # Galería con filtros + lightbox
│   ├── proceso/page.tsx   # Timeline del proceso
│   ├── sobre-nosotros/    # Historia y valores
│   └── servicios/
│       ├── cocinas/
│       ├── banos/
│       ├── electricos/
│       ├── fontaneria/
│       ├── integral/       # 🆕 Remodelación completa
│       ├── pisos/          # 🆕 Porcelanatos, cerámicos
│       ├── pintura/        # 🆕 Acabados decorativos
│       └── consultoria/    # 🆕 Asesoría profesional
└── components/
    ├── HeroSection.tsx         # Hero con Spline 3D + partículas
    ├── Navbar.tsx              # Sticky con dropdown animado
    ├── Footer.tsx              # Footer completo con links
    ├── ServicesGrid.tsx        # Grid de servicios con hover
    ├── StatsSection.tsx        # Contadores animados
    ├── BeforeAfterSection.tsx  # 🆕 Slider antes/después
    ├── Interactive3DSection.tsx # 🆕 Sección 3D dedicada
    ├── ProcessSection.tsx      # Timeline del proceso
    ├── TestimonialsSection.tsx # Testimonios de clientes
    ├── FAQSection.tsx          # FAQs con acordeón animado
    └── WhatsAppButton.tsx      # Botón flotante
```

## ✨ Features

- **Hero 3D interactivo** con Spline
- **30 partículas animadas** flotando en el fondo
- **Orbes de gradient** dinámicos con movimiento
- **Texto cíclico animado** (Cocinas → Baños → Electricidad...)
- **100% responsive** (móvil, tablet, desktop)
- **Dark mode premium** con acentos en rojo brand
- **Efecto glassmorphism** en tarjetas y navbar
- **Botón glow animado** en CTAs
- **Marquee de confianza** infinito
- **Contadores animados** que cuentan al hacer scroll
- **Slider antes/después** interactivo (drag)
- **Sección 3D dedicada** para explorar modelos
- **Lightbox** en galería
- **Formulario de contacto** con redirect a WhatsApp
- **FAQ acordeón animado**
- **Transiciones de página** suaves entre rutas

## 🏃‍♂️ Desarrollo

```bash
npm install
npm run dev        # http://localhost:3001
npm run build      # Production build (requiere 4+ GB RAM)
npm run start      # Production server
```
